# slideshop

Slideshop.my ialah marketplace untuk menjual design slide PowerPoint secara online.

## Konsep Projek

Platform ini membolehkan pelanggan:

- melihat pakej slide
- memilih slide secara individu pada harga **RM1 per slide**
- atau membeli **full bundle / pakej**
- membuat bayaran secara online
- menerima softcopy slide melalui email secara automatik selepas bayaran berjaya

## Matlamat MVP

Fasa pertama projek memberi fokus kepada:

- homepage
- katalog pakej
- halaman detail produk
- cart
- checkout
- integrasi payment gateway
- penghantaran digital automatik

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## Status

Projek ini sedang dibangunkan sebagai MVP untuk Slideshop.my.
